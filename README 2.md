
# 需求
因为个人做了一个培训班，所以需要一个记录学生课时的工具。因为需求很简单，只是简单的增加、扣除课时而已，在网上找的收费的感觉没必要，免费的又不适用，于是自己弄了个微信小程序。

# 架构
前端：uni-app
后端：gin-vue-admin

# 开发工具

Visual Studio Code、微信开发者工具、MySQL

# 目录结构

gin-vue-admin
├── server (后端服务)
└── web（后端页面）
└── wx（微信小程序）
